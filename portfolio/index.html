My Projects

Project 1 (IOT Smart Parking Using RFID.)

The IoT Smart Parking System using RFID integrates RFID technology and IoT to streamline parking management. RFID tags are attached to vehicles and detected by RFID readers installed at parking entry and exit points. This system provides real-time data on parking space availability, allowing users to quickly find and reserve spots through a mobile app or web interface. By automating space tracking and updating availability in real-time, this solution reduces the time spent searching for parking and enhances overall efficiency in parking facilities.

Project 2 (The IoT Weather Reporting System')

The IoT Weather Reporting System uses sensors to track weather conditions like temperature, humidity, and wind speed. These sensors send the collected data to a central server or cloud platform through communication modules. Users can access real-time weather information via a mobile app or website, making it easy to stay updated on current weather conditions. This system helps provide accurate weather reports and can be used for various applications such as agriculture, event planning, and outdoor activities.
